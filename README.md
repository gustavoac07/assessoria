# 🌿 ELO Assessoria Alimentar - Site Responsivo

Site profissional para consultoria em segurança alimentar, desenvolvido com HTML, CSS e JavaScript puro.

## 📁 Arquivos

- **index.html** - Arquivo HTML completo com CSS e JavaScript integrados
- **logo.png** - Logo da ELO (coloque na mesma pasta do index.html)

## 🚀 Como Usar

### 1. No VSCode
1. Abra o VSCode
2. Crie uma pasta para o projeto
3. Copie os arquivos `index.html` e `logo.png` para a pasta
4. Clique com botão direito no `index.html` e selecione "Open with Live Server"
5. O site abrirá no navegador automaticamente

### 2. Sem Live Server
- Simplesmente abra o arquivo `index.html` diretamente no navegador (duplo clique)

## 🎨 Estrutura do Site

- **Header**: Navegação fixa com logo, menu responsivo e botão WhatsApp
- **Hero Section**: Chamada principal com dois botões (WhatsApp e Saiba Mais)
- **Sobre**: Apresentação da missão e visão da empresa
- **Valores**: 6 cards com os valores principais
- **Serviços**: 6 cards descrevendo os serviços oferecidos
- **Footer**: Informações de contato, redes sociais e horário comercial
- **WhatsApp Flutuante**: Botão fixo no canto inferior direito

## 📱 Responsividade

O site é totalmente responsivo e funciona perfeitamente em:
- Desktop (1200px+)
- Tablet (768px - 1199px)
- Mobile (até 767px)

## 🔧 Personalizações

### Alterar Telefone/WhatsApp
Procure por `5511999999999` no arquivo e substitua pelo seu número (com código do país)

### Alterar Email
Procure por `contato@eloassessoria.com.br` e substitua pelo seu email

### Alterar Cores
No CSS (dentro de `<style>`), procure por:
- `--verde-escuro: #2D5016;` - cor primária
- `--dourado: #D4AF37;` - cor de destaque
- `--branco: #FFFFFF;` - cor de fundo

### Alterar Redes Sociais
Procure por `facebook.com`, `instagram.com` e `linkedin.com` no footer e substitua pelos seus links
